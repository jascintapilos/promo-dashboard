# How Claude Automates IGMP Back Office Promo Management

## The Problem
**Before:** Manual data entry into IGMP BO for every promotion
- Copy-paste promo details into 5+ forms per brand
- Create, update, activate each one individually
- Risk of human error (typos, missed fields, wrong dates)
- **Time:** 30–60 mins per promotion request

## The Solution: Claude Agent

```
📊 Spreadsheet Request (P###)
        ↓
    [Claude reads specs]
        ↓
🤖 Claude Agent Logs Into IGMP BO
        ↓
  [Creates promo via API]
        ↓
✅ Activates & Verifies
        ↓
📋 Reports Success (with proof)
```

## What Claude Does
1. **Reads** your promotion request (code, amount, dates, rules)
2. **Logs in** to IGMP Back Office automatically
3. **Creates** the promo record (Deposit Bonus / Free Credit / Free Spins)
4. **Updates** details if needed
5. **Activates** it to go live
6. **Confirms** it worked by reading back the saved data

## The Benefit
| Before | After |
|--------|-------|
| 30–60 mins per promo | ~30 seconds per promo |
| Manual clicking | Zero clicking |
| Error-prone | Verified & logged |
| One brand at a time | Multi-brand in parallel |
