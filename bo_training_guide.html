📚 BO Training Guide for Sales Team

Learn to Query Promos, Check Eligibility, and Assign Bonuses Independently

Module 1: Login & Navigation

Learning Objective

Get comfortable accessing BO and finding the promo section.

1 Access BO Portal

Module 2: Query Promos by Brand

Learning Objective

Find and verify if a promo code exists and is ACTIVE.

QPRO1–19 All Bonus Types Menu 3.2 – Promotion Codes

1 Access Promotions List
For QPRO/QP2: Go to Menu 3.2 → Promotion Codes
For WS1/WS2: Go to iGMP 3.1 (Deposit) or 3.3 (Free Credit)
You'll see a table of all promos for your brand
2 Search for a Promo Code
Use the search/filter box: Type the promo code (e.g., FT_DEP_VIP_100)

3 Check Status Column
Look for Status column. It should show:
✅ ACTIVE = promo is live
⏳ UPCOMING = not yet live
❌ INACTIVE or ARCHIVED = can't use
💡 Tip: If you can't find a promo code, ask in #ba-promo. It might not exist or might be on a different brand.

Module 3: Check Promo Details

Learning Objective

Read promo terms (bonus amount, eligible tiers, valid dates, turnover).

Specific Menu Locations

  • QPRO/QP2: Menu 3.2 → Click on Promo Code → Settings icon
  • WS1/WS2: iGMP 3.1/3.3 → Select Bonus → View Details
1 Open Promo Details
QPRO/QP2: In Menu 3.2, click the Settings icon under "Action" column
WS1/WS2: In iGMP 3.1/3.3, select the bonus to view all details
Details page opens showing all promo info
2 Check Key Fields
Look for and note:
  • Bonus Amount — MYR/SGD/currency + amount (e.g., "100% up to MYR 500")
  • Eligible Tiers/Categories — which player tiers can use (Bronze, Silver, Gold, etc.)
  • Valid From / Valid Until — dates (is it still valid today?)
  • Turnover — (e.g., 8x, 12x, 20x)
  • Min Deposit — minimum deposit required
3 Evaluate Red Flags
Ask yourself:
  • Is TO >20x? (might be hard to clear)
  • Is min deposit >player's usual deposit?
  • Is bonus amount too low (<10% of deposit)?
  • Has the promo expired already?
⚠️ Red Flag: If you spot anything suspicious, DON'T offer it. Post in #ba-promo with the question.

Practice Scenario 2

Task: Open FT_DEP_VIP_100 and identify:

  • What's the bonus amount?
  • Which tiers are eligible?
  • What's the turnover?
  • Any red flags?

Module 4: Check Player Claim History

Learning Objective

Verify if a player has already claimed a promo recently.

1 Access Player Records
Go to: Players or Members section
Search for player by ID/name/email
2 Open Player Profile
Click on the player name
Look for tab: Claim History or Bonuses Claimed
3 Check If Promo Was Claimed
Search claim history for promo code
If you see it listed with a date, they've already claimed it
Decision: If claimed recently (same month), don't offer again
💡 Tip: If unsure about claim rules, post in #ba-promo asking "Can this player claim this promo again?"

Practice Scenario 3

Task: Check if player "John123" has claimed FT_DEP_VIP_100 before.

Steps: Go to Players → Search "John123" → Open profile → Check claim history → Report findings

Module 5: Assign Promos (WS1/WS2/QPRO1/QP2A)

Learning Objective

Assign a verified promo to a player in BO/CRM.

For WS1/WS2 (CRM/Smartico)

1 Open Player CRM Record
Go to Smartico or CRM for your brand
Search for player
2 Find Promo/Bonus Section
Look for: Assign Bonus, Add Promo, or Bonuses tab
3 Enter Promo Code
Type promo code (e.g., FT_DEP_VIP_100)
System should auto-populate the bonus details
4 Confirm & Save
Review assignment details
Click Confirm or Save
Player receives the bonus

For QPRO1/QP2A (BO Dashboard)

1 Go to Assignment Section
In BO, find: Promotion Assignments or Manual Assignment
2 Select Player & Promo
Enter: Player ID, Promo Code
System validates both exist
3 Assign & Verify
Click Assign
System confirms: "Assignment successful"
Note the assignment ID for records
⚠️ Common Error: If assignment times out or fails, try again. If it persists 3+ times, post in #ba-promo with error message.

Quick Reference Sheet

Before Offering Any Bonus (Tier 1 Checklist)

Check Where to Look What You're Looking For
Promo exists? Promotions List Search by code → Found = YES
Is ACTIVE? Promotions List Status = "ACTIVE"
Player tier matches? Promo Details Check eligible tiers vs player tier
Not expired? Promo Details Valid Until date > today
Already claimed? Player Claim History Promo code NOT in recent claims
Red flags? Promo Details TO <20x, Min Dep reasonable, Amount >10%

Practice Exercises (Hands-On)

Exercise 1: Complete Promo Lookup (15 min)

Scenario: Sales VM wants to offer a Gold player promo GLD_DEP_150.

Your Tasks:

  1. Find the promo in BO
  2. Verify it's ACTIVE
  3. Check bonus details (amount, TO, eligible tiers)
  4. Verify player tier matches
  5. Check player hasn't claimed recently
  6. Spot any red flags
  7. Decide: Offer or Escalate?

Exercise 2: Complete Assignment (15 min)

Scenario: You've decided to offer promo FT_DEP_VIP_100 to player "SandboxPlayer".

Your Tasks:

  1. Query BO to confirm promo is ACTIVE
  2. Check player claim history (no recent claims)
  3. Assign the promo in BO/CRM
  4. Verify assignment successful
  5. Document the assignment ID

Exercise 3: Spot Red Flags (10 min)

Scenario: You find promo TEST_DEP_RISKY with these terms:

  • Amount: 100% up to MYR 1000
  • Turnover: 25x
  • Eligible: Gold only
  • Status: ACTIVE

Question: What red flags do you see? Would you offer this?

Answer: TO is 25x (>20x threshold) = very hard to clear. Post in #ba-promo before offering.

Troubleshooting

Problem Likely Cause Solution
"Promo not found" Code doesn't exist or is on different brand Ask in #ba-promo: "Is [code] active?"
"Assignment times out" BO lag or network issue Retry 1–2 times. If persists, post in #ba-promo
"Player not found" Wrong player ID or spelled wrong Double-check player ID in CRM
"Permission denied" Your account lacks assignment rights Contact Jascinta for access upgrade
"Can't see player claim history" Different menu path per brand Ask trainer where claim history is for your brand
