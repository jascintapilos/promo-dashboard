## Version control rule

When a deliverable changes:

1. Replace only the authoritative project copy.
2. Record the change and approval in the project decision log when material.
3. Keep raw/source files outside this deliverables folder.
4. Re-run the relevant visual, formula, and data-quality checks before executive
   circulation.
