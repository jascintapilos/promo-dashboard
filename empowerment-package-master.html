🎯 Section 2: Decision Framework

Overview

This framework gives you clear guidance on what decisions you can make INDEPENDENTLY. It assumes you have BO access and basic training. It breaks decisions into 3 tiers:

  • Tier 1: You decide alone using BO (80%+ of questions) — Immediate
  • Tier 2: Post in #ba-promo Slack (policy/operational decisions) — 2–4 hours
  • Tier 3: Escalate to @Wai Yip (strategic only) — Rare

✅ TIER 1: You Decide Alone (BO-Trained)

You have BO access. Check everything yourself. No escalation needed for these operational decisions.

Step 1: Check Promo in BO

Before offering any bonus, navigate to the correct menu:

  • QPRO/QP2: Go to Menu 3.2 (Promotion Codes)
  • WS1/WS2: Go to iGMP 3.1 (Deposit), 3.3 (Free Credit), or 3.15 (Free Spins)

Then verify:

  • Query BO for promo code → Is it ACTIVE?
  • Check eligible tiers → Does player's tier match?
  • Check validity dates → Is it still valid (not expired)?
  • Check claim history → Has player already claimed recently?

Result: ✅ All checks pass? Offer it. ❌ Any fail? Go to TIER 2.

Step 2: Evaluate Bonus Terms

Check This Red Flag? Action
Turnover >20x? Yes → hard to clear Ask in #ba-promo if OK
Min deposit >player's usual? Yes → they won't use Ask in #ba-promo if OK
Free credit <10% deposit? Yes → seems low Ask in #ba-promo if OK

Step 3: Which Bonus Type Fits?

Scenario Best Bonus
Player says "want to play longer" → Deposit Bonus
Low balance / at-risk player → Free Credit
Slot player / specific game lover → Free Spins

Step 4: Assign in BO

Once you've verified the promo and checked terms:

  • WS1/WS2: Assign in CRM directly
  • QPRO1/QP2A: Assign in BO Dashboard
  • Other brands: Check Routing Matrix → post in #ba-promo if needed

You own this end-to-end. No waiting on promo team for routine assignments.

❓ TIER 2: Ask in #ba-promo (2–4 Hours)

Post these in #ba-promo Slack (NOT to Wai Yip's DMs). Expected response: 2–4 hours during work hours (10 AM–7 PM).

Policy & Operational Decisions

Request Type What To Ask Example
Create New Promo "Can we create [bonus type] with [terms]?" "Create 100% deposit bonus up to MYR 1000, 8x TO, valid 7 days"
Exception Request "Can we lower TO / extend date?" "Reduce TO to 6x for this VIP?"
Bulk Assignment "Can we assign [promo] to [group]?" "Assign FT_DEP_100 to top 50 VIP players"
Red Flag "This bonus seems [risky]. Is it OK?" "MYR 500 with 25x TO—sustainable?"

How to Post in #ba-promo

Format: [Request Type] | Context | Question
Example: "Exception Request | QPRO1 Gold VIP | Can we lower TO from 8x to 6x for FT_DEP_VIP_100?"

🔴 TIER 3: Escalate to @Wai Yip (Strategic Only)

Very rare. Tag @Wai Yip in Slack #ba-promo for high-level strategy decisions only.

Strategic Escalation Triggers

  • Promo Cost / ROI: "This bonus costs too much—should we revise the structure?"
  • Strategic Updates: "Should we pause deposits and focus on Free Spins for Q3?"
  • High-Level Decisions: "Should we invest more in VIP retention promos?"

5 Shared Principles (Culture)

These guide decisions for BOTH Sales and Promo teams:

1. Player Protection First

Every bonus should be sustainable for the player—not a trap. Reasonable TO (8–12x is standard). Clear terms.

2. Transparency

Players should understand the bonus before claiming. "100% up to MYR 500, 8x turnover, valid 7 days" = clear.

3. Speed & Autonomy

Move fast without bottlenecks. If it fits the framework, offer it. Don't wait for approval on every offer.

4. Continuous Learning

If you don't understand something, ask in #ba-promo (not DMs). Share answers with the team.

5. Compliance Always Wins

If there's any risk (fraud, regulation, player harm), escalate. Don't try to figure it out alone.

Sign-Off Checklist

Before using this framework, verify you can:

  • Access BO and query promo details (codes, terms, eligible tiers, validity dates)
  • Check player tier in BO/CRM and match to promo eligibility
  • Review claim history to verify player hasn't already claimed
  • Identify red flags in bonus terms (high TO, low amounts, etc.)
  • Assign promos in BO/CRM for WS1/WS2/QPRO1/QP2A
  • Make Tier 1 decisions independently (no waiting on promo team)
  • Post Tier 2 requests in #ba-promo (not DMs to Wai Yip)
  • Understand the 5 Shared Principles

By signing this, you commit to: (1) completing BO training, (2) making decisions independently, and (3) using #ba-promo for policy questions only.

📚 Section 5: BO Training Guide

Overview

Sales team learns to query promos, verify eligibility, and assign bonuses. 5 modules, 1 day (4–6 hours).

Learning Outcomes

  • ✅ Log into BO for each platform (QPRO, QP2, WS1, WS2)
  • ✅ Query promos by brand and code
  • ✅ Read promo details (tiers, terms, validity)
  • ✅ Check player claim history
  • ✅ Assign promos independently (WS1/WS2/QPRO1/QP2A)
  • ✅ Spot red flags (expired promos, tier mismatch, high TO)

5 Training Modules

🔓 Module 1: Login & Navigation

Duration: 30–45 min | Goal: Learn how to access each platform

Content: Credentials, login steps, main menu structure for QPRO/QP2/WS1/WS2

Practice: Log into each platform; navigate to Promotion Codes menu

🔍 Module 2: Query Promos by Brand

Duration: 1–1.5 hours | Goal: Find active promos

Content: Filter by brand/merchant, read promo lists, check status (active/inactive)

Practice: Query QPRO1 for all Deposit bonuses; query WS1 for Free Credit promos

📋 Module 3: Check Promo Details

Duration: 1 hour | Goal: Verify promo terms & eligibility

Content: Read detail page (amount, TO, min deposit, eligible tiers, validity dates, quota)

Practice: Open 3 promos; spot red flags (expired, high TO, no quota)

📊 Module 4: Check Player Claim History

Duration: 45 min | Goal: Avoid duplicate claims

Content: Find claim history, read claim records, make claim/no-claim decisions

Practice: Check 5 players; determine if they can claim again

✅ Module 5: Assign Promos in BO

Duration: 1–1.5 hours | Goal: Assign bonus end-to-end

Content: Navigate to assign form, fill in player details, submit, confirm

Practice: Assign 3 bonuses (WS1, QPRO1, QP2A); capture confirmations
