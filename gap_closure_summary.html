Root Causes

  1. No Decision Framework → Sales doesn't know what decisions are OK to make alone
  2. No Bonus Mechanics Primer → Sales lacks mental model of how bonuses work
  3. No Weekly Promo Export → Sales doesn't have reference list; always asks "does this exist?"
  4. Unclear Brand Assignment Matrix → Sales doesn't know who assigns for QPRO2–19, WS3, etc.
  5. No Async Channel → All questions go to Wai Yip; no Slack Tier 2 flow
  6. No Culture Bridge → Sales and Promo teams operating independently

The Solution: 4-Document System

4 Strategic Documents

A complete system to transform decision-making, from bottleneck to autonomy:

1️⃣ Sales Team Lead Decision Framework

Solves: Root causes #1, #2

  • Tier 1: Decisions sales can make alone (80% of routine Qs)
  • Tier 2: Questions for #ba-promo Slack (2–4 hr response)
  • Tier 3: Strategic escalations to Wai Yip only
  • Plus: 5 Shared Principles (player protection, transparency, speed, learning, compliance)

2️⃣ Bonus Assignment Routing Matrix

Solves: Root cause #4

  • Self-Serve: WS1, WS2, QPRO1, QP2A
  • Promo Team: QPRO2–19, QP2B/C/D, others
  • Plus: Decision tree + request format

3️⃣ Weekly Active Promo List Template

Solves: Root cause #3

  • Updated every Monday 10:30 AM
  • All active promos by brand & bonus type
  • Sales never asks "does this exist?"

4️⃣ Implementation Guide

Solves: Root cause #5 + provides rollout

  • 4-week phased rollout (Jun 12 – Jul 9)
  • Success metrics & risk mitigation
  • Coaching log to track learning
